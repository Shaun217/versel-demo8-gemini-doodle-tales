# 🎨 DoodleTales (Gemini Edition)

利用 Google Gemini 的多模态视觉能力，将孩子随手画的涂鸦瞬间变成一个温馨的睡前故事。

## ✨ 核心功能
1.  **看懂涂鸦**：无需文字描述，Gemini 能直接“看”懂孩子画的是什么（奥特曼、小猫、还是一个奇怪的怪兽）。
2.  **故事生成**：结合家长输入的简单情节（如“迷路了”），自动续写完整童话。
3.  **纯前端实现**：无需服务器，直接在浏览器调用 Google API，保护隐私。

## 🛠 如何使用
1.  **获取 Key**：
    - 访问 [Google AI Studio](https://aistudio.google.com/)。
    - 点击左上角 **Get API key**。
    - 这是一个免费且大方的 API（个人使用完全免费）。
2.  **部署**：
    - 将 `index.html`, `style.css`, `script.js` 拖入 GitHub。
    - Vercel 自动部署。
3.  **体验**：
    - 打开网页 -> 填入 Key。
    - 用手机拍一张孩子的画 -> 上传。
    - 写一句话灵感 -> 点击生成！

## ⚠️ 注意事项
- 本项目使用 `gemini-1.5-flash` 模型，它既便宜（免费）又快，且支持图片输入。
- API Key 仅存储在你的浏览器内存中，刷新页面即消失，不会上传到任何第三方服务器。
